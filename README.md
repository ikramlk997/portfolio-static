# Portfolio de Ikram Lakhal (Version Statique)

Ce dépôt contient la **version statique** de mon portfolio personnel.  
La version complète a été développée en **Laravel**, mais pour la publication rapide sur GitHub Pages, j'ai exporté une version HTML/CSS/JS statique.

---

## Aperçu

Le site statique contient les sections suivantes :

- **Accueil** : Présentation et introduction.
- **À propos** : À propos de moi, mes compétences et mes intérêts.
- **Compétences** : Mes compétences techniques (HTML, CSS, JavaScript, PHP, Laravel, UML, etc.).
- **Expérience & Diplômes** : Stages et formations.
- **Projets** : Mes projets phares avec lien vers le code ou les démonstrations.
- **Contact** : Formulaire de contact (simulation) et email direct.

> ⚠️ Note : Certaines fonctionnalités dynamiques (comme la gestion réelle du formulaire de contact ou la récupération de projets depuis un CMS Laravel) ne sont pas incluses dans cette version statique.

---

## Technologies utilisées

- HTML5 / CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Font Awesome (icônes)
